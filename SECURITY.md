# 安全策略

## 支持版本

当前项目处于早期开源阶段，默认只维护 `main` 分支上的最新版本。安全修复会优先合入 `main` 并在 changelog 中记录。

## 报告漏洞

如果你发现安全问题，请不要先公开发布复现细节。可以通过 GitHub Security Advisory 或私下联系维护者报告。

报告时请尽量包含：

- 受影响的版本或提交
- 复现步骤
- 可能影响的数据或权限范围
- 你建议的修复方向

## 安全边界

Visual Worktree 是本地桌面应用，会读取用户配置的仓库目录、Git 状态和本地任务记录。项目设计目标是不上传这些数据。任何新增网络能力都应在 PR 中说明用途、数据范围和关闭方式。

未签名的本地构建不代表官方发布包。请只从可信来源下载应用，并在需要分发给更多用户时自行完成代码签名和公证。
